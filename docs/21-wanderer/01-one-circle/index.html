Built the wanderer as class wrapping a vehicle instead of extending it. Not sure why, but that is what felt right to me. Perhaps is the way the vehicle code is sitting in an apparatus in the illustrations of the original paper by Craig Reynolds.

Wanderer's point to seek is a location on a circle as described in the paper. The next seek point is selected by drawing a second dot on a second circle and using that location to find a new location on the arc contained by that circle. Press the "l" key while running to freeze the apparatus in place. I don't believe the blue dot is the actual next seek point, but it is where this seek point was relative to it's predecessor since the apparatus is being drawn after the update happens.
